## 🌟 About The Project

Freelancer Platform is a modern web application built to connect freelancers and clients in one place.  

The platform allows users to create professional profiles, post projects, communicate with each other, and collaborate efficiently. The main goal of this project is to provide a smooth, scalable, and user-friendly freelancing experience using modern web technologies.

This project also helped in improving full stack development skills, UI/UX design understanding, authentication systems, and real-world project building experience.

## 🚀 Features

- 🔐 User Authentication (Login / Signup)
- 👨‍💻 Freelancer Profile Creation & Management
- 📢 Project & Job Posting
- 💬 Real-Time Communication
- ⭐ Rating and Review System
- 🔍 Search & Filter Functionality
- 📱 Fully Responsive Design
- 🌙 Dark & Light Mode Support
- ⚙️ Admin Dashboard
- 🔒 Secure Backend Integration

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/ShaluPandey-12/Freelancing.git
```

### Navigate to Project Folder

```bash
cd Freelancing
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
